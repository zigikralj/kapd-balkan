# KapdBalkan website

This is the website for Kraljevsko Akademsko Prirodnjačko Društvo - Balkan (Royal Academic Society of Naturalists - Balkans).

The website is hosted on [kapd-balkan.rs](https://www.kapd-balkan.rs).

The website is built with [Astro 7.0](https://astro.build/blog/astro-7/)  framework and it is a static site.
